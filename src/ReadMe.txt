## English ##
system.ini file configuration parameters
- Exclude: Identifies the tables that do not want to be sent from Source to Destination.
- Exclude: table_name[column_name_1:column_name_2] identifies columns that are not intended to be sent from source to destination.
- Include: Specifies the tables to be synchronized from destination to source.
